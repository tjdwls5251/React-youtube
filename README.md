React와 tailwindcss 로 구현한 유튜브 클론 코딩.
Page 링크 : https://merry-frangollo-3b53c9.netlify.app

구현 상태 : 
          메인페이지 -> /
                       Header ( Logo & Search ) 
                       Body (인기 영상 이미지 & 타이틀 & 업로드된 시간 )
          
          검색페이지 -> /videos/${search} 
                      Body 검색한 검색어를 기반으로 관련된 영상을 보여줌.
          
          상세페이지 -> /videos/watch/${videoId} 
                      Body 좌측 (영상 & 타이틀 & 설명) 
                      Body 우측 (클릭된 영상과 관련된 영상 이미지 & 타이틀 )
